Pinkinite

What is this mod?

The Pinkinite mod adds a new set of building blocks to the game. It is extremely similar to quartz, but as you could probably tell from its name, it isn't white.

Why should I use it?

If you've ever been annoyed at the lack of colourful building blocks that have stairs then this mod is for you. If not, why are you still here?
